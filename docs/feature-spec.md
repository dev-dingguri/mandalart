# Mandalart — 기능 명세서 (v1.3.4)

> 소스코드 기반으로 역추출한 기능 명세서입니다.

---

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| **제품명** | Mandalart |
| **버전** | 1.3.4 |
| **목적** | 하나의 목표를 81칸(9×9) 격자로 분해하여 구체적인 실행 계획을 수립하는 목표 설정 도구 |
| **플랫폼** | 웹 (모바일 최적화 포함) |
| **지원 언어** | 한국어, English, 日本語, 中文(简体) |

---

## 2. 데이터 모델

```
만다라트 1개 = 메타(제목) + 토픽 트리
토픽 트리 = 1개 루트 → 8개 하위 목표 → 각 8개 실행 항목 = 총 73개 노드
```

| 제약 | 값 |
|------|-----|
| 최대 만다라트 수 | 20개 |
| 제목 최대 길이 | 50자 |
| 토픽 텍스트 최대 길이 | 50자 |
| 격자 크기 | 9×9 (3×3 그리드 × 9) |

---

## 3. 사용자 모드

### 3.1 게스트 모드 (미로그인)

| 항목 | 내용 |
|------|------|
| **데이터 저장** | 브라우저 localStorage |
| **만다라트 개수** | 1개 (임시) |
| **제한** | 새 만다라트 생성 불가, 다른 기기 동기화 불가 |
| **라벨** | 제목 위에 "(이 기기에만)" 표시 |
| **삭제** | "삭제" 대신 "초기화" 제공 |

### 3.2 인증 모드 (로그인)

| 항목 | 내용 |
|------|------|
| **데이터 저장** | Firebase Realtime Database |
| **만다라트 개수** | 최대 20개 |
| **동기화** | 실시간 양방향 동기화 (Firebase `onValue` 구독) |
| **로그인 방식** | Google OAuth (팝업) |

### 3.3 게스트 → 인증 전환 (데이터 마이그레이션)

- 로그인 시 게스트 데이터를 Firebase로 자동 업로드
- **첫 번째 만다라트만** 마이그레이션됨
- `sessionStorage` 플래그로 중복 업로드 방지

## 4. 핵심 기능

### 4.1 만다라트 CRUD

| 기능 | 진입점 | 동작 | 비고 |
|------|--------|------|------|
| **생성** | 빈 화면 중앙 버튼 / 좌측 드로어 하단 버튼 | 빈 만다라트 생성 후 자동 선택 | 게스트 불가, 최대 20개 |
| **선택** | 좌측 드로어 목록 클릭 | 해당 만다라트로 전환 | 전환 중 빈 화면 방지 처리 |
| **삭제** | 좌측 드로어 → 항목 "..." 메뉴 → 삭제 | 확인 다이얼로그 후 삭제 | 메타+트리 원자적 삭제 |
| **초기화** | 좌측 드로어 → 항목 "..." 메뉴 → 초기화 | 확인 다이얼로그 후 내용 비움 | 구조 유지, 텍스트만 제거 |
| **이름 변경** | 드로어 "..." 메뉴 → 이름 바꾸기 / 제목 클릭 | 텍스트 입력 다이얼로그 | 최대 50자 |

### 4.2 토픽 편집

- 격자의 각 셀 클릭 → 텍스트 입력 다이얼로그
- 최대 50자, 초과 시 경고 + 저장 버튼 비활성화
- Enter로 저장, Shift+Enter로 줄바꿈
- 저장 즉시 Firebase 동기화 (인증 모드)

### 4.3 뷰 모드

| 모드 | 설명 | 전환 방법 |
|------|------|-----------|
| **전체 보기 (All View)** | 9×9 전체 격자 표시 | 기본값, 축소 아이콘 클릭 |
| **확대 보기 (Focus View)** | 선택한 3×3 그리드 확대 | 확대 아이콘 클릭 |

### 4.4 스와이프 네비게이션 (Focus View 전용)

- 터치 스와이프로 인접 그리드 이동
- **상하좌우 + 대각선** 방향 지원
- 플릭 제스처 감지: 500ms 이내 빠른 스와이프 시 속도 가중치 적용 (최대 5배)
- 스와이프 임계값: 컨테이너 너비 × 35%
- 이동 실패 시 원래 위치로 smooth 스크롤 복원

## 7. UI 구조

### 7.1 헤더

```
[메뉴 아이콘] ─── [Mandalart 제목] ─── [아바타/로그인] [설정("...")]
```

- **아바타**: 사용자 이름 첫 글자 (원형 배지), 클릭 시 이름/이메일/로그아웃 드롭다운
- **미로그인 시**: "로그인" 텍스트 버튼

### 7.2 좌측 드로어 — 만다라트 목록

- vaul Drawer (왼쪽에서 슬라이드)
- 만다라트 목록 (최신순)
- 현재 선택 항목 강조
- 각 항목 "..." 메뉴: 이름 바꾸기 / 삭제(또는 초기화)
- 하단 "새 만다라트" 버튼

### 7.3 우측 드로어 — 설정

- vaul Drawer (오른쪽에서 슬라이드)
- 테마, 언어, OSS 라이선스, 버전, 연락처

### 7.4 다이얼로그

| 다이얼로그 | 용도 | 특징 |
|------------|------|------|
| **TextInputDialog** | 제목/토픽/이름 편집 | 글자 수 카운터, 초과 경고, Enter 제출 |
| **ConfirmDialog** | 삭제/초기화 확인 | 커스텀 확인 텍스트 |
| **AlertDialog** | 에러 알림 | ESC/외부 클릭 차단, 강제 확인 |
| **SignInDialog** | 로그인 유도 | Google 로그인 버튼 + 안내 문구 |

## 9. 라우팅

| 경로 | 페이지 |
|------|--------|
| `/{lang}` | 메인 페이지 |
| `/{lang}/oss` | 오픈소스 라이선스 (lazy load) |
| `*` | `/{lang}`으로 리다이렉트 |

언어 감지 우선순위: URL 경로 > localStorage > 브라우저 설정 > 기본값(en)

## 10. 분석 이벤트 (Firebase Analytics)

| 이벤트 | 시점 |
|--------|------|
| `trackAppVersion` | 앱 초기화 |
| `trackUserType` | guest/authenticated 판별 |
| `trackSignIn` / `trackSignOut` | 로그인/로그아웃 |
| `trackMandalartCreate` / `Delete` / `Reset` | 만다라트 CRUD |
| `trackGuestUpload` | 게스트 데이터 마이그레이션 |
| `trackViewModeChange` | 전체/확대 뷰 전환 |
| `trackLanguageChange` / `trackThemeChange` | 설정 변경 |

## 11. 알려진 제한사항

- **저장 성공 피드백 없음**: Firebase 실시간 구독으로 UI가 즉시 반영되므로 별도 토스트 없음
- **게스트 마이그레이션**: 첫 번째 만다라트만 이관, 나머지는 유실
- **오프라인 지원 없음**: 네트워크 연결 필수
